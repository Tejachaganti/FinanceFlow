# 💰 FinanceFlow Backend

FinanceFlow Backend is a secure RESTful API powering the FinanceFlow personal finance management platform. Built with Node.js, Express.js, and MongoDB Atlas, it provides authentication, expense tracking, budget management, financial analytics, and profile management features.

## 🚀 Features

### 🔐 Authentication & Security

* JWT-based Authentication
* Protected Routes
* Secure Password Handling
* User Authorization

### 💵 Expense Management

* Add Expenses
* Update Expenses
* Delete Expenses
* Expense Categorization
* Expense History Tracking

### 📊 Budget Management

* Create Budget Plans
* Budget Monitoring
* Spending Tracking
* Budget Performance Analysis

### 📈 Financial Analytics

* Expense Insights
* Spending Trends
* Category-wise Analysis
* Financial Summary Reports

### 👤 User Profile Management

* User Profile Information
* Account Settings
* Personal Finance Preferences

### ☁️ Cloud Infrastructure

* MongoDB Atlas Database
* Render Deployment
* RESTful API Architecture

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Tejachaganti/financeflow-backend.git
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Run Production Server

```bash
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
CLIENT_URL=*
```
