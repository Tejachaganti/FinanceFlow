FinanceFlow

AI-Powered Personal Finance Management Workspace

This repository has been upgraded from a beginner expense tracker into a professional split-stack fintech project with React, Tailwind, Express, MongoDB, JWT auth, analytics, exports, and AI-style insights.

Open README

Frontend

Modern React dashboard in frontend/ with Tailwind CSS, Framer Motion, Recharts, protected routing, and reusable finance state.

  • Dashboard analytics
  • Expense CRUD and filters
  • Budget and savings management
  • Profile, theme, and exports

Backend

Production-style Express API in backend/ with MongoDB models, JWT auth, uploads, exports, and AI insight generation.

  • Auth and protected routes
  • Expenses, budgets, profile APIs
  • Analytics and report exports
  • Receipt and avatar uploads

Run Locally

Backendcd backend && npm install && npm run dev

Frontendcd frontend && npm install && npm run dev

Copy each .env.example to .env before starting.
