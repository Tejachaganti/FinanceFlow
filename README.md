## вҡҷпёҸ Local Setup

### Clone Repository

```bash
git clone https://github.com/Tejachaganti/financeflow-backend.git
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Quick Launch

```bash
run-app.bat
```
